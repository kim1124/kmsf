# Backend Base Template

`templates/backend-base`는 향후 backend scaffold를 위한 placeholder template이다.

현재 `create-kmsf` production catalog에는 연결되어 있지 않다. backend app, API server, database migration, deployment contract가 확정된 뒤 scaffold source로 승격한다.

## 현재 상태

- 구조 고정용 placeholder
- 생성 경로 아님
- 앱 본체 구현 대상 아님
