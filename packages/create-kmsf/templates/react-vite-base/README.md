# {{project_name}}

이 프로젝트는 `create-kmsf`에서 생성한 React/Vite starter다.

## 시작

```bash
npm install
npm run dev
```

브라우저에서 <http://localhost:5173>을 연다.

## 포함 항목

- Vite
- React
- React Router Data Mode
- Zustand
- i18next/react-i18next
- Axios
- Zod
- Vitest
- Playwright

## 주요 스크립트

| Script | 설명 |
| --- | --- |
| `npm run dev` | Vite dev server |
| `npm run build` | TypeScript와 production build |
| `npm run lint` | ESLint |
| `npm run test:run` | Vitest |
| `npm run test:e2e` | Playwright |
